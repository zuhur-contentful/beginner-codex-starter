# beginner-codex-starter

A starter template and step-by-step guide for building real software with AI — from zero experience to a working app.

---

## Start here

**Step 1 — Download this repo**

1. Click the green **Code** button at the top of this page
2. Click **Download ZIP**
3. Unzip it — you'll get a folder on your computer

> Already have Git? `git clone https://github.com/zuhur-contentful/beginner-codex-starter.git`

**Step 2 — Open the guide**

1. Open the folder
2. Find **`guide.html`**
3. Double-click it — opens in your browser
4. Everything is in there: what to install, what to learn, how to build

> Save as PDF: **File → Print → Save as PDF** in your browser
